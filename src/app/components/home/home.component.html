Bonjour! C'est VisiteurProfil Application

Cette Application vous permettra de consulter la liste des utilisateurs et leurs profil détaillés

Comment ca marche ? veuillez suivre les étapes suivantes :

  • Pour consulter la liste des utilisateurs : cliquer sur le boutton "Utlisateurs" Utlisateurs

  • Pour consulter un profil détaillé avec la liste des dépôts : saisir le nom utilisateur dans la barre de recherche
  • Pour consulter les statistiques d'un depots: une fois sur le profil choisissez un repository et cliquer sur statistiques
